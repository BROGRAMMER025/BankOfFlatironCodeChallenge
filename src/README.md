# Bank Transaction Project

Author: Allan Murigi

## FEATURES

Details for banking transactions
- Date
- Description
- Category
- Amount

It includes a search bar to look up transactions

It includes a table to display the looked up transactions

- React.js
- CSS
- Local JSON data (db.json)
- Canvas
- Stackoverflow
